Cómo funciona el Box Model

Cada elemento HTML es una "caja" que puede contener contenido, relleno, borde y margen. En este proyecto verás ejemplos de cómo se comporta cada parte y cómo box-sizing cambia el cálculo de tamaño.

Scuderia

Este bloque usa padding, border y margin. Gracias a box-sizing: border-box, el ancho total se mantiene constante.

Red Shift

Prueba a cambiar el padding y el border en el archivo styles.css y verás cómo no afecta el tamaño total de la caja.

Ejemplo: box-sizing: content-box

Esta caja usa el modelo predeterminado (content-box), por lo que el padding y el border aumentan su ancho total. Mide con herramientas de desarrollador para comprobarlo.

Comparativa: border-box vs content-box

border-box

El ancho total siempre es el mismo aunque cambies padding y border.

content-box

El ancho total aumenta al añadir padding o border.

Puedes abrir las herramientas de desarrollador (F12) y observa el cálculo de la caja en cada ejemplo.
